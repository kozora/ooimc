## OOI MineCraft Server

#### 服务器配置

- Intel Xeon L5630 处理器
- 32GB DDR3 REG ECC 内存
- 120GB SSD 硬盘
- 电信/联通 双线网络

#### 规则

- 尊重他人
- 不随意捣乱
- 不懂的请尽量自行查阅百科


#### 插件、命令及定价

部分命令使用会产生金钱消耗，金钱来源来自在线奖励，此外无其他来源

- 使用```/ptt```查看自己在线时间
- 使用```/ptt ac```领取在线奖励（每15分钟可领一次奖励，每次奖励5$）
- 使用```/home```传送回家，计费算法：每次使用消耗3$
- 使用```/bal```查看自己的余额

#### 客户端下载

本服采用1.12纯净服，任何原版1.12的客户端均可进入游戏，群文件有打包文件
